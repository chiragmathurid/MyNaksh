## ▶️ Running the App (Expo)

Follow these steps to run the app locally using Expo:

### 1. Install dependencies

```bash
npm install
```

### 2. Install Expo CLI (if not installed)

```bash
npm install -g expo-cli
```

### 3. Start the development server

```bash
npx expo start
```

### 4. Run the app

After starting the server, you have multiple options:

* 📱 **On your phone (recommended):**

  * Install **Expo Go** from Play Store / App Store
  * Scan the QR code shown in the terminal or browser

* 🤖 **On Android Emulator:**

  ```bash
  a
  ```

  (Press `a` in the terminal)

* 🍎 **On iOS Simulator (Mac only):**

  ```bash
  i
  ```

  (Press `i` in the terminal)

### 5. Editing the app

* Open the project in your code editor
* Make changes → app reloads automatically

---

### ⚠️ Notes

* Make sure your phone and computer are on the same network
* If QR scan doesn't work, try switching to **Tunnel** mode in Expo DevTools
